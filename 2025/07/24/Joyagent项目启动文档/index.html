Joyagent项目启动文档

项目地址: https://github.com/jd-opensource/joyagent-jdgenie/tree/data_agent

运行环境:

  • 操作系统: Linux Mint 22.1
  • JDK版本: 17.0.16
  • Maven版本: 3.9.11
  • Node.js版本: v22.17.1
  • Pnpm: 7.33.1
  • python版本: 3.11

准备阶段:

  1. 安装Java

  2. 安装Maven: https://dlcdn.apache.org/maven/maven-3/3.9.11/binaries/apache-maven-3.9.11-bin.tar.gz

  3. 安装Node: https://nodejs.org/en/download

  4. 安装pnpm:

  5. python安装: https://www.python.org/downloads/

  6. 注册deepseek开放平台(提供大模型服务)

  7. 注册serper(提供搜索服务)

  8. 克隆仓库 git clone https://github.com/jd-opensource/joyagent-jdgenie.git

启动项目:

  1. 进入项目后执行

  2. 根据提示安装缺少的依赖

  3. 配置后端文件, 位于 genie-backend/src/main/resources/application.yml

    • 将所有<input llm server here>替换成https://api.deepseek.com/v1

    • 将所有<input llm key here>替换成https://platform.deepseek.com/api_keys创建的密钥

    • llm:
    model: deepseek-chat
    max-tokens: 8192
